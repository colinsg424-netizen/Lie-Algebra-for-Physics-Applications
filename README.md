# Lie Algebra for Physics Applications
A Python framework for construction and analysis of finite-dimensional Lie algebras using matrix representations. The project is designed for applications in theoretical physics, such as quantum mechanics and gauge theories.

It provides tools to compute Lie brackets, structure constants, generators, adjoint representations, Killing forms, Casimir operators, and Lie group elements via the exponential map. An example of the wide implementations of this framework is included demonstrating quantum spin procession using the su(2) Lie algbera.
